# Orchestral Documentation

이 문서는 Orchestral 프로젝트의 공식 문서입니다.

## 로컬에서 확인하기

```bash
# Docsify CLI 설치
npm install -g docsify-cli

# 문서 서버 실행
docsify serve docs

# 브라우저에서 http://localhost:3000 접속
```

## 문서 구조

- `guide/` - 사용 가이드
- `architecture/` - 아키텍처 문서
- `api/` - API 레퍼런스
- `advanced/` - 고급 주제
- `misc/` - FAQ, 기여 가이드 등
